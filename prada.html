Type + Brand Design

The Devil Wears Prada

A high-fashion promotional system built around a custom typeface, poster redesign, and supporting campaign materials.

Role

Typeface design, poster design, promotional system, art direction

Tools

Illustrator · Photoshop

Deliverables

Custom typeface · Poster · Ticket · Pamphlet · Paper bag

Project Description

Typography as fashion language.

This project reimagines promotional materials for The Devil Wears Prada through the lens of luxury fashion branding. The custom typeface became the core visual device, giving the campaign a distinct editorial personality.

The system uses dramatic scale, sharp forms, and refined composition to create a tone that feels elegant, recognizable, and connected to the film's fashion world.

  • Developed a custom typeface inspired by high-fashion editorial references.
  • Created a cohesive set of promotional materials.
  • Used typography as the main driver of brand recognition.
